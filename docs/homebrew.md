# Installing via Homebrew

```bash
brew tap g-dos/gedos
brew install gedos
```

## Updating

```bash
brew upgrade gedos
```

## Uninstalling

```bash
brew uninstall gedos
brew untap g-dos/gedos
```

## Installing from HEAD (latest dev)

```bash
brew install g-dos/gedos/gedos --HEAD
```

## Troubleshooting

- If Ollama is not found: `brew install ollama`
- If ffmpeg is not found: `brew install ffmpeg`
- If Python version conflicts: `brew install python@3.12`
